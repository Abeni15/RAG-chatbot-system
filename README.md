# 🤖 RAG Chatbot System

This project implements a complete Retrieval-Augmented Generation (RAG) pipeline using LangChain, FAISS, and HuggingFace models.

## 🚀 Features
- Sentence-based chunking (NLTK)
- Embedding using Sentence Transformers
- FAISS vector database for retrieval
- History-aware conversational system
- Gradio-based user interface
- Multimodal support (OCR for image text extraction)

## 🧠 Architecture
Document → Chunking → Embedding → Vector Store → Retrieval → LLM → Response

## 🛠️ Technologies Used
- Python
- LangChain
- FAISS
- HuggingFace Transformers
- Gradio
- NLTK
- Tesseract OCR

## 💡 Example Use Cases
- Document-based Q&A
- AI-powered assistants
- Knowledge retrieval systems

## 📌 How to Run
1. Open the notebook in Google Colab
2. Install dependencies
3. Upload your document
4. Run all cells
